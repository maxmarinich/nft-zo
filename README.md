# nft-zo

### install
```
npm ci
```
### setup
- rename `secrets.dev.json` to `secrets.json` and fill required fields

### Development / Deploy testnet:

```
npm run start
npm run compile

npm run local:deploy
npm run local:mint
npm run local:transfer
```

### Development / Deploy mainnet:

```
npm run start
npm run compile

npm run deploy
npm run mint
```

### Development / Run client dev server

```
npm run serve
```
